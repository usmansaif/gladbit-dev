Full Stack Developer (Laravel + Vue.js)

Develop full-stack web applications using Laravel and Vue.js, ensuring clean, scalable, and secure code.

Full-time Remote Flexible Hours Work From Anywhere

About the Role

We’re looking for a passionate and skilled Full Stack Developer (Laravel + Vue.js) to join our growing GladBit team. You’ll collaborate with talented engineers, designers, and product managers to build reliable and scalable software used by clients across the globe.

Key Responsibilities

  • Develop scalable full-stack web applications using Laravel and Vue.js.
  • Collaborate with designers, frontend, and backend teams to deliver high-quality features.
  • Write clean, secure, and maintainable code following best practices.
  • Integrate APIs and manage database structures efficiently.
  • Participate in code reviews and agile sprint planning.

Requirements

  • Proficiency in Laravel, Vue.js, and RESTful API development.
  • Experience with MySQL, Git, and Docker.
  • Strong understanding of OOP, MVC, and modern development standards.
  • Familiarity with unit testing and CI/CD pipelines.
  • Excellent problem-solving and communication skills.

How to Apply

Excited to work with us at GladBit? Send your resume and a short cover letter to:
career@gladbit.com

Please include the job title “Application for Full Stack Developer (Laravel + Vue.js)” in your email subject line.
