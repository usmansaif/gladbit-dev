Cosmo Camp

STEM Camp Registration Portal | Education | Arizona, USA

About the Project

Cosmo Camp is an online portal built for Cosmosphere—a renowned space museum and STEM education center in Hutchinson, Kansas. The portal enables parents to register their children for STEM-based summer camps, manage camper profiles, and complete payments in a seamless manner.

As the lead developer, I oversaw the complete development cycle including requirement analysis, database design, R&D, backend and frontend development, and deployment. The platform supports both parent-facing and admin-side operations with rich features for profile management, payments, and camp session assignments.

Features for Parents

  • - Register child(ren) as camper(s)
  • - Profile & roommate preferences
  • - Medical & allergy details submission
  • - Payment gateway & discount system
  • - Referral code and scholarship application
  • - Dashboard for multiple profiles & camp history
  • - Direct support and communication tools

Features for Admin

  • - Event and session management
  • - Camper tracking & profile access
  • - Sales, payments, and refund processing
  • - FAQ, CMS content, uAsk bulletin board
  • - Reporting tools & permission-based access

Technology Used

PHP Laravel MySQL Bootstrap jQuery GitHub VS Code AWS
