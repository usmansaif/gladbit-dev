PakPoultry.pk – Daily Rate SMS Alert System

Automated poultry price alerts by SMS, powered by data scraping and Laravel | Multan, Pakistan

About the Project

PakPoultry.pk is a subscription-based poultry information service that delivers customized daily SMS alerts to its users. I built the entire platform from the ground up using Laravel, PHP, jQuery, and Bootstrap. It’s designed to automate the collection, organization, and distribution of poultry rate data from various sources.

Key Features

  • - Automated SMS Alerts: Personalized daily poultry rate updates sent to subscribers by area and alert type.
  • - Data Scraping & Categorization: Intelligent scripts fetch and organize poultry prices from multiple online sources.
  • - Dynamic Rate Lists: Automatically generates categorized rate sheets for chicks, eggs, chicken, and farms—also shared to Facebook pages.
  • - Admin Panel: Full-featured admin interface to manage SMS/email servers, subscribers, and scraped data.
  • - End-to-End Execution: Independently developed all backend and frontend components.

Technology Used

Laravel, PHP, MySQL, Bootstrap, jQuery, SMS API Integration, PHPStorm
