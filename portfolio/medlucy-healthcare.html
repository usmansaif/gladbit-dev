MedLucy – Healthcare Management Software

Modern medical management solution built for clinics across Luxembourg | Europe

About the Project

MedLucy is a next-generation healthcare platform designed for Luxembourg’s medical sector. The software streamlines clinical workflows, centralizes patient data, and empowers healthcare professionals with modern tools for scheduling, billing, and collaborative care. I contributed to developing its robust admin panel, enhancing system performance, and integrating key features to improve the day-to-day operations of clinics.

Key Features

  • - Real-time Agenda: Manage waiting rooms and appointments with live updates.
  • - Complete Patient File: Centralized access to medical history, documents, and prescriptions.
  • - Timeline View: Visualize patient history in an intuitive interface.
  • - Lab Request Module: Streamlined process for initiating and tracking lab tests.
  • - Automated Forms: Replace manual forms with fully digital workflows.
  • - Billing System: Fast, clean, and efficient billing with automation support.
  • - Patient Referrals: Facilitate doctor-to-doctor referrals effortlessly.
  • - Prescription Tools: Compliant medication management tailored to Luxembourg’s standards.

Technology Used

Laravel PHP Vue.js Vuex Vuetify MySQL GitHub
