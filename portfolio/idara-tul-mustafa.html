Event Scheduling Platform

Custom Event Scheduler for Religious Scholars | Idara Tul Mustafa | Gujranwala, Pakistan

About the Project

The Event Planner application is a custom-built solution for Idara Tul Mustafa, developed to streamline the process of scheduling religious events for their scholars. The system automates manual bookings and introduces a powerful calendar system to eliminate double-bookings and streamline communication between scholars and organizers.

Key Features

  • - Scholar Profiles: Each scholar has a personalized calendar and event history.
  • - Event Scheduling: Real-time booking system with instant updates.
  • - Conflict Prevention: Automatically detects and prevents overlapping bookings.
  • - Feedback System: Post-event organizer ratings to foster accountability.
  • - Admin Dashboard: Full control over scholar schedules and event visibility.

Technology Used

PHP Laravel Bootstrap MySQL Apache Linux PHPStorm GitHub
